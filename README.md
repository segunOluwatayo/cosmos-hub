# CosmosHub // NASA Mission Archive

A full-stack NASA data intelligence terminal built with React + Node.js/Express.
Classified-mission-terminal aesthetic. AI-powered analysis via Claude. Real NASA data.

## Tech Stack

- Frontend: React 18, Vite, Tailwind CSS
- Backend: Node.js, Express
- AI: Anthropic Claude
- Deployment: Vercel (frontend) + Render (backend)

## Local Setup

### Prerequisites

- Node.js ≥ 18
- NASA API Key — free at api.nasa.gov
- Anthropic API Key — console.anthropic.com

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
